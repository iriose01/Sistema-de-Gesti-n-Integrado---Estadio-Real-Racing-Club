# Sistema-de-Gesti-n-Integrado---Estadio-Real-Racing-Club
Sistema híbrido para el estadio del Racing con red redundante MikroTik y VPN a AWS. Gestiona 16 TPVs sincronizados con RDS y CCTV IP integrado con Kinesis para monitorización remota. Dashboard React para control total de ventas y seguridad. Arquitectura de alta disponibilidad.
